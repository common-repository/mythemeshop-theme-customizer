Enhance your OnePage Lite theme with extra functionality through sections like: Buttons, Clients, Counter, Features, Blog Posts, Services, Team, Testimonials & Tweets.

== Description ==

Use this plugin to enable drag and drop homepage sections in <a href="https://wordpress.org/themes/onepage-lite/">OnePage Lite</a> theme.

**Note**: Currently this plugin works only with OnePage Lite theme.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `mythemeshop-theme-customizer` folder to the to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can see new widgets available in the 'Widgets' page in the dashboard.
4. Add these widgets in homepage widget area in any order.

== Frequently Asked Questions ==

= Where is the settings page for this plugin? =

This plugin don't have any settings page, you can configure all the sections and settings from 'Widgets' page.
